# PapaCarlo

```bash
papa_carlo new_project
```

Will create following structure:

* new_projet
    * config
        * options
            * config.yml
        * boot.rb
    * lib
        * new_project
        * new_project.rb
    * spec
        * helpers
        * new_project
        * spec_helper.rb
    * Rakefile
    * Gemfile
